# Calculadora Web

Projeto de uma calculadora web desenvolvida com HTML, CSS e JavaScript, com foco na prática de lógica de programação, manipulação do DOM e eventos.

## Descrição
Aplicação de calculadora que permite realizar operações matemáticas básicas por meio de uma interface simples e intuitiva.  
O projeto foi desenvolvido para reforçar conceitos fundamentais do desenvolvimento front-end e organização de código em JavaScript.

## Funcionalidades
- Operações básicas: soma, subtração, multiplicação e divisão
- Suporte a números decimais
- Uso de parênteses em expressões matemáticas
- Botão para limpar o display
- Botão para apagar o último caractere
- Execução do cálculo ao clicar no botão de igual (=)
- Validação básica de expressões inválidas

## Tecnologias Utilizadas
- HTML5 — estrutura da aplicação
- CSS3 — estilização e layout da interface
- JavaScript — lógica da calculadora, eventos e manipulação do DOM
